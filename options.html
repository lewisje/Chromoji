Emoji Polyfill options

For now these options don't work, you'll just see textual emoji everywhere regardless of domain or Private Use Area status.

Some custom fonts use characters from the Private Use Area. This option makes sure we don't break sites with such fonts, but keep in mind that some emojis will not be shown when this is turned on (e.g. SoftBank).

A list of domains that should not display emoji. This should be the last part of any domain seperated with commas "," e.g. google.com will include images.google.com, le.com will include google.com too as it ends in le.com (so be careful).
